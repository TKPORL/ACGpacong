## 坑提醒(下个对话注意)
- **单 HTML 在 `file://` 下 fetch 不到 JSON**(浏览器 CORS 限制),必须 http:// 或部署 Pages
- **Playwright 测试用 python 内置 http.server** 模拟 http:// 验证 JSON 加载
- **PowerShell curl 输出带 UTF-16 BOM**,测 API 用 python urllib
- **Flask 5000 端口常被占**,自动改 5001
- **测试 Flask 进程用 Start-Process 启动,后续要 Stop-Process 清理**
- **Playwright Firefox 不支持 `is_mobile=True`**,用 chromium 测移动端
- **Playwright Webkit 没装会报错**,需要时 `playwright install webkit`(较大,可选)
- **Cloudflare Pages 必须选 `单html版本` 为 Build output directory**,否则 JSON 404
